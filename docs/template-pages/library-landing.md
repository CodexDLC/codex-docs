# Library Landing Template

Canonical source file: `docs/templates/library-landing-template.md`

Use this template for the long-form package landing page that lives inside each library docs site.

## Required blocks

- one-line package definition;
- role in the ecosystem;
- install;
- quick start;
- recommended paths;
- subsystem map;
- API reference entry;
- changelog and migration links;
- related libraries.

## Notes

- This is the canonical long-form package surface.
- README and PyPI should derive from this narrative.

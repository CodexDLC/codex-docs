# Library Metadata Template

Canonical source file: `docs/templates/library-metadata-template.yml`

Use the metadata template to register or update a library in the ecosystem catalog.

## Required fields

- `library_id`
- `display_name`
- `role`
- `maturity_status`
- `latest_docs_url`
- `supported_version_lines`
- `related_libraries`
- `readme_source`
- `landing_owner`
